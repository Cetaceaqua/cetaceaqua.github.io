Blender 3D 入门随手记 第1天

发表于 2021-05-17 更新于 2021-05-18

写在前面

早先在B站看到这个转载的“Blender Introduction Series for Beginners”系列视频，心里痒痒了好久，一直想学 Blender，但奈何高中学习任务重，抽不出来时间。

这次恰逢一周的综合实践，我又生病在家，正好就把在收藏夹里埋藏以久的这套教程扒出来学一学。

附上第一集的视频链接，有兴趣的话也可以跟着学学。

随手记

第一集的内容正如它的副标题“Absolute Basics”，对 Blender 的界面和基础操作进行了详细介绍，并穿插了两个低模建模小任务。

（为了更方便的跟随教程操作，我没把 Blender 调成中文。）

首选项

将鼠标选中改为右键，空格键操作改为搜索。

快捷键

（省略了滚轮放大缩小和 Ctrl + Z 之类早就刻在DNA里的快捷键。）

G：移动

S：缩放

R：旋转

G + X / Y / Z：仅沿 X / Y / Z 方向移动

S + X / Y / Z：仅沿 X / Y / Z 方向缩放

R + X / Y / Z：仅沿 X / Y / Z 方向旋转

Alt + G：重置移动

Alt + S：重置缩放

Alt + R：重置旋转

Ctrl + A：应用菜单

（重置会使物体回到上一次应用变换时的默认状态。）

Shift + A：添加菜单

Shift + 鼠标中键：平移视口

Numpad 1：正视图

Numpad 7：顶视图

Numpad 3：右视图

Ctrl + Numpad 1：后视图

Ctrl + Numpad 7：底视图

Ctrl + Numpad 3：左视图

Numpad .：专注选中物体

/：专注选中物体

（与 Numpad . 不同的是 / 会使其他物体隐藏，再按一次 / 以取消隐藏其他物体。）

B：框选

（使用 鼠标中键 拖动框选可以取消选择。）

A：全选

（双击 A 取消全部选择。）

Shift + S：对齐菜单

Ctrl + Space：最大化编辑器区域 / 退出最大化编辑器区域

Ctrl + J：合并物体

Shift + D：复制物体

Numpad 0：摄像机视角

Ctrl + Alt + 0：把当前视图设为摄像机视角

属性

Properties -> Scene Properties -> Units -> Length：更改使用的长度单位

特性

鼠标左键 单击放置的 3D 游标将成为添加的新物体的中心。

鼠标左键 拖动编辑器区域圆角外的区域来复制或合并编辑器区域。

调整添加的物体的数值的菜单仅在进行其他操作前可用。

（还没进行其他操作，比如移动或是变形的话，可以按 F9 唤出调整菜单。）

切换变换坐标系到本地可以使物体按自己的坐标系进行变换。

切换变换原点来更高效的旋转物体。

切换吸附模式来更高效的放置物体。

黄点是物体的原点，可以通过物体菜单来设置原点。
