关于

自打去年知道了 Hexo + Github Pages 能免费搭出效果挺不错的静态博客，就一直想给自己弄一个。

结果还是懒癌发作，整整拖了一年多才正儿八经弄好。

（顺便整了个自定义域名。）

其实吧，折腾了半天整这个博客，最大的需求其实是不想让太多人看到我写的东西。

是不是听起来有点怪？🥴

主要原因是我对自己挺没自信的，要是用Tumblr、Lofter、博客园之类的博客网站写东西，博文很有可能会被人无意中刷到，万一正好那篇博文我排版烂、文笔差、顺带代码再出错几处，岂不是尴尬的要死。

但如果你是专程访问了我的博客站来读博文的话，说明不管我的博文多烂，你也是有目的性的来读的，这样能让我减轻不少可能坑到别人的负罪感。🙃

（从这里开始才是关于页面该有的内容。）

关于

你好啊，我是鲸流🐳 / cetaceaqua。

我喜欢游戏开发🎮、插画👾和拟人化动物（Furry）🐉，所以博客上的大部分博文都会围绕这几个主题展开。

我通常在Twitter和哔哩哔哩上活跃，如果喜欢我的创作内容，可以考虑关注我💗。

我唯一稍有知名度的领域是独立游戏 UNDERTALE ❤的同人创作。

环境配置

（虽然我没啥名气，也没人过问我的配置，但写这个貌似能增加些关于页面的逼格。）

（都不是很高端的设备 / 软件，贵的我也买不起，所以反倒对大部分人都有一定的参考价值。）

数位屏 / 数位板：

  • 绘王 Kamvas Pro 16（GT-156Pro） （更新的产品已经支持绘王笔头几乎不晃动的硬笔了，有点心动。）

  • 绘王 Inspiory HC16 （便宜的很。）

  • iPad Air 4 （拿来记笔记的时间远比画画的时间长。）

绘画软件：

  • Clip Studio Paint （春季打折时候半价购入 Pro 版，我是聪明逼。👏）

  • Aseprite （像素画软件中性价比高且功能够强的存在，等1.3更新图块编辑器之后应该就能宣判 Pyxel Edit 的死刑了。）

  • Inkscape （免费的矢量绘画软件，1.0版本更新后界面更好看了。）

  • Procreate （常年位居 App Store 创意类 App 榜首，拿它画画简直是享受）
